GRUPO 1
Porcentajes en Costos de Producción
Calcular costo total, costo variable y su representación porcentual
% porcentajeC(x) = Cf+Cv
// Datos del ejercicio
EMPRESA: "Café Andino S.A."
Costo fijo: $300
Costo variable por unidad: $8
Modelo: C(x) = 8x + 300
// Qué deben calcular
  • El costo total para producir 40 sacos de café
  • El costo variable total a 40 sacos
  • Qué porcentaje del costo total representa el costo variable
  • El costo promedio por saco a 40 unidades
// Paso a paso
1
Sustituye x = 40 en C(x) = 8x + 300
2
Calcula Cv = 8 × 40
3
Usa %Cv = (Cv / C(40)) × 100
4
Divide C(40) / 40 para el costo promedio
// Fórmulas que necesitas
C(x) = Cf + Cv·x
%Cv = (Cv / Ctotal) × 100
Cp = C(x) / x

Precios, Márgenes de Ganancia y Comisiones
Calcular precio de venta, margen y comisión de un vendedor
MargenComisión
// Datos del ejercicio
TIENDA "TecnoMóvil"
Costo del producto: $180
Margen de ganancia deseado: 35%
Comisión del vendedor: 4% sobre venta
Unidades vendidas: 15
// Qué deben calcular
  • El precio de venta con el margen del 35% aplicado sobre el costo

GRUPO 4
Ecuaciones Lineales aplicadas
Hallar el punto de equilibrio entre costo e ingreso
Ec. linealPunto equilibrio
// Datos del ejercicio
EMPRESA "Dulces Quito"
Costo: C(x) = 4x + 600
Precio de venta por unidad: $10
Ingreso: I(x) = 10x
// Qué deben calcular
  • El punto de equilibrio — la cantidad x donde Costo = Ingreso
  • El costo y el ingreso en ese punto (deben ser iguales)
  • La utilidad si se venden 150 unidades
  • ¿Cuántas unidades hay que vender para una utilidad de $500?
// Paso a paso
1
Iguala: 4x + 600 = 10x
2
Despeja x: pasa los términos con x a un lado
3
Para la utilidad: U(x) = I(x) − C(x) = 10x − (4x+600) = 6x − 600
4
Para U=500: resuelve 6x − 600 = 500
// Fórmulas que necesitas
C(x) = I(x) → punto de equilibrio
U(x) = I(x) − C(x)
