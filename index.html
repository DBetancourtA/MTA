// Objetivo
Aplicar operaciones matemáticas a contextos empresariales reales: cálculo de precios, márgenes, funciones de costo e ingreso y modelado matemático.
// Metodología
Cada semana incluye casos prácticos resueltos paso a paso, gráficas interactivas y un taller por grupos con clave de acceso para practicar en clase.
// Herramientas
Las gráficas se generan directamente en el navegador — sin instalar calculadoras ni software extra. Solo abre la página y trabaja.
